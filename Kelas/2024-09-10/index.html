Belajar HTML dan CSS

HTML di gunakan untuk membuat tampilan website, CSS digunakan untuk mengatur tampilan

Materi HTML

  1. h1 untuk judul
  2. p untuk paragraf
  3. img untuk gambar
  4. Table untuk tabel
  5. input untuk form
  6. li untuk list atau daftar

Materi CSS

  • Margin,Padding,Position untuk mengatur jarak antar elemen html
  • display untuk mengatur elemen html agar tampil atau tidak tampil, mendatar(Kolom),atau menurun (Baris)
  • Style untuk mengatur font
